# Промпт: готовая рамка с дыркой → прозрачный PNG с полупрозрачными тенями

> Для случая, когда на образце ДЫРКА УЖЕ ВЫРЕЗАНА (центр пустой/белый),
> а вокруг листа и по внутреннему краю есть тени.
> Фото вырезать не нужно — задача чисто про альфа-канал.

---

## ТЕКСТ ПРОМПТА (копировать целиком)

```
Сделай из приложенной картинки прозрачную PNG-рамку (RGBA).
На образце ДЫРКА УЖЕ ВЫРЕЗАНА (центр пустой) — НЕ детектируй и не вырезай фото.

ЗАДАЧА — собрать альфа-канал:
1. БУМАГА (лист с рваными краями, перфорацией, линовкой/клеткой/полями) —
   полностью НЕПРОЗРАЧНАЯ, показывай как есть.
2. БЕЛОЕ — и фон СНАРУЖИ листа, и пустая ДЫРКА внутри — полностью ПРОЗРАЧНОЕ.
3. ТЕНИ — и наружная (вокруг листа), и внутренняя (по краю дырки) —
   ПОЛУПРОЗРАЧНЫЙ ЧЁРНЫЙ, ровно настолько, насколько пиксель темнее белого.
   Так тени работают на любом фоне (и затемняют его одинаково внутри и снаружи).

КАК ОТДЕЛЯТЬ (ключевой признак — цвет, а не яркость):
- Бумага ТЁПЛАЯ (R−B заметно > 0, кремовая). Белое и серые тени — НЕЙТРАЛЬНЫЕ (R≈G≈B).
  → Маска бумаги = «тёплые» пиксели (R−B выше порога, ~10).
  Если бумага не тёплая (белая/в клетку) — отделяй её по ТЕКСТУРЕ
  (бумага текстурная; белое и тени гладкие).
- Уровень белого возьми по самым ярким пикселям (≈99-й перцентиль яркости).

ПОСТРОЕНИЕ:
- Маску бумаги почисти: морфологическое «закрытие» (соединить тонкие линии сетки),
  крупнейшая связная область. ВАЖНО: центральную дырку НЕ заливать —
  заполняй только МЕЛКИЕ внутренние дырки от текстуры (по площади), большую дырку оставь.
- Альфа тени (для НЕбумажных пикселей): alpha = clip(1 − яркость/белое, 0..1).
  Чистое белое → 0 (прозрачно), серая тень → частично. Цвет тени — чёрный (0,0,0).
- Собери: бумагу (её родной цвет, альфа=1, мягкий край ~1px) положи ПОВЕРХ
  слоя-тени; белое остаётся прозрачным.
- НИКАКИХ дополнительных ореолов/обводок: альфа = либо бумага, либо тень, либо 0.

ПРОВЕРКИ (обязательно покажи):
1. Маска бумаги (что непрозрачно).
2. Итог на ШАХМАТНОМ фоне (видно прозрачность дырки и фона + полупрозрачные тени).
3. Итог на ЦВЕТНОМ фоне (тени должны естественно затемнять цвет и внутри, и снаружи).

Сохрани как PNG с альфа-каналом.
```

## Параметры, которые можно крутить
| Скажи | Что изменится |
|---|---|
| «тени плотнее / светлее» | множитель альфы теней (например `clip(k·(1−ярк/белое))`, k≈1.3 плотнее) |
| «тени убрать совсем» | оставить только бумагу: белое и серое → полностью прозрачно |
| «бумага белая/в клетку» | отделять лист по текстуре, а не по тёплому оттенку |
| «край бумаги жёстче / мягче» | размытие маски бумаги (0 px жёстко … 1.5 px мягче) |

## Почему так надёжно
- Не нужно ничего «угадывать» про фото — дырка уже на образце.
- Тень переводится в честный полупрозрачный чёрный по формуле `1 − ярк/белое`,
  поэтому на белом выглядит как оригинал, а на любом другом фоне затемняет его корректно.
- Бумага отделяется по контрасту «тёплая бумага / нейтральные фон-и-тени»
  (или по текстуре, если бумага белая) — это устойчивее, чем пороги по яркости.

## Когда использовать ЭТОТ промпт, а когда другой
- ЭТОТ — если на образце **дырка уже вырезана** (центр пустой). Самый простой и надёжный.
- Промпт «вырезать окно по фото» (`notebook_frame.md`) — если в рамку **вставлено фото**,
  и его нужно убрать, сделав на его месте прозрачное окно.
