# Промпт: прозрачная PNG-рамка «тетрадный листок» из картинки-образца

> Прикладываешь картинку-образец и вставляешь текст ниже.
> Картинка нужна ТОЛЬКО как образец — из неё извлекается рамка, а не копируется фото.

---

## ТЕКСТ ПРОМПТА (копировать целиком)

```
Сделай из приложенной картинки прозрачную PNG-рамку (RGBA) для наложения
поверх фото. Картинку используй ТОЛЬКО как образец (нужна рамка, а не само фото).

═══════════════════════════════════════════════════════════
ПАРАМЕТРЫ (значения по умолчанию; меняй, если попрошу)
═══════════════════════════════════════════════════════════
- Растворяемые края: определить АВТОМАТИЧЕСКИ по тени (см. ниже).
- Ширина растворения: 60–120 px, пропорционально глубине тени.
- Окно (дыра под фото): определить автоматически по фото.
- Ширина акварельной каймы окна: средняя.

═══════════════════════════════════════════════════════════
РЕЗУЛЬТАТ
═══════════════════════════════════════════════════════════
1. Лист бумаги с рваными краями. Сохрани ВСЁ, что есть на образце
   (перфорацию от пружины, линовку/клетку, поля, надписи, каракули).
   Фон вокруг листа — полностью прозрачный.
2. На месте фотографии — сквозное ПРОЗРАЧНОЕ окно с мягкими «акварельными»
   краями (рваными, плавно растворяющимися в прозрачность). Окно должно
   ПОЛНОСТЬЮ перекрывать фото — от исходного снимка не должно остаться
   ни кусочка.
3. Отдельную тень НЕ рисуй. Вместо неё мягко растворяй в прозрачность те края
   листа, где на образце лежала тень (плавный градиент альфы от края внутрь).
   Остальные края оставь чёткими рваными.

═══════════════════════════════════════════════════════════
ТЕХНИКА
═══════════════════════════════════════════════════════════
[Отделение листа от фона]
- Отделяй бумагу от фона по ТЕКСТУРЕ и ЯРКОСТИ: фон гладкий и однотонный,
  бумага текстурная (линовка/клетка/зерно). Фон срезай заливкой от краёв
  картинки (flood fill) по маске «яркое И гладкое».
- НЕ опирайся на тёплый оттенок — бумага бывает и кремовой, и белой, и в клетку.
- Маску листа почисти: крупнейшая связная область + заливка внутренних дыр.
- Никаких полупрозрачных «ореолов» вокруг листа: альфа чёткая везде,
  кроме специально растворяемых краёв.

[Окно под фото]
- Зону фото ищи как крупный СПЛОШНОЙ цветной/тёмный/холодный блок.
  Тонкие надписи, сетку и мелкие каракули отсекай (морфологическое «открытие»
  + крупнейшая связная область).
- ВНИМАНИЕ: светлое небо/блёклые части фото могут сливаться с бумагой.
  Поэтому определи полный ПРЯМОУГОЛЬНИК, гарантированно покрывающий ВСЁ фото
  (включая небо), и проверь его визуально, наложив на образец.
- Окно строй так: сплошная сердцевина по этому прямоугольнику (полностью
  покрывает фото) + рваный край НАРУЖУ из многомасштабного шума со смещением,
  широкой мягкой растушёвкой, лёгкой полупрозрачной каймой и зернистостью.

[Авто-определение растворяемых краёв по тени] — ДО удаления фона:
- Возьми яркость «чистого» фона (углы картинки).
- Для каждой из 4 сторон листа замерь УЗКУЮ полосу фона сразу снаружи края
  (≈4–30 px) и возьми её ТЁМНЫЙ перцентиль (~20-й) — это тень.
  (Среднее по широкой зоне не годится: мягкая тень тонет в шуме.)
- depth = яркость_фона − тёмный_перцентиль_стороны.
- Сторону РАСТВОРЯЙ, если depth > 5 И depth > 0.45 × max(depth по сторонам).
  Ширину делай пропорционально глубине (~60–120 px: глубже тень → шире).
- Остальные стороны (обычно верх) оставляй ЧЁТКИМИ.
- Если тени нет нигде (фронтальный свет) — дефолт: низ + одна боковая.

[Растворение края]
- Для выбранных краёв: по каждой строке/столбцу найди границу листа и плавно
  гаси альфу к этому краю на заданную ширину (мягкий градиент в прозрачность).

═══════════════════════════════════════════════════════════
ПРОВЕРКИ (обязательно покажи)
═══════════════════════════════════════════════════════════
1. Силуэт листа (фон отделён).
2. Найденный прямоугольник окна, наложенный на образец.
3. Отчёт авто-определения краёв: по каждой стороне depth и решение
   (растворять/чёткий, ширина).
4. Итоговый PNG на ШАХМАТНОМ фоне (проверка прозрачности).

Сохрани результат как PNG с альфа-каналом.
При моих правках меняй ТОЛЬКО нужные параметры, остальное не трогай.
```

## Шпаргалка по быстрым правкам

| Что сказать | Что изменится |
|---|---|
| «окно чуть меньше / больше» | размер прозрачной дыры |
| «акварельную кайму уже / шире» | мягкость края окна |
| «растворение краёв сильнее / слабее» | ширина затухания краёв (px) |
| «растворяй низ и право» (вручную) | отключить авто-определение, задать края явно |
| «верхний край тоже растворить» | добавить сторону к растворению |
| «оставь все края чёткими, без растворения» | рамка без затухания (только рваный лист) |
| «верни обычную тень вместо растворения» | отдельный полупрозрачный серый слой-тень |

## Заметки (почему так)
- Бумага отделяется по ТЕКСТУРЕ, а не по цвету — поэтому работает и для белой/клетчатой.
- Окно строится «с запасом» (сплошная сердцевина покрывает фото), чтобы от старого
  снимка не осталось краёв; «акварельность» добавляется только наружу.
- Растворяемые края берутся из реальной тени на образце → выглядит естественно
  и не плодит «второй листок» (как делала бы нарисованная мягкая тень).
